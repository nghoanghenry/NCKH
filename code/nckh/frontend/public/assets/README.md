# Hướng dẫn thêm ảnh cho các loài

Để hiển thị đầy đủ thông tin, bạn cần thêm các file ảnh sau vào thư mục `public/assets/`:

## Danh sách ảnh cần thiết:

1. **te-te-java.webp** - Tê tê (Pangolin)

   - Đã có sẵn

2. **vooc.jpg** - Voọc chà vá chân nâu (Red-shanked Douc)

   - Tìm ảnh voọc chà vá với bộ lông nhiều màu

3. **khi-voi.jpg** - Khỉ vọi (Gibbon)

   - Tìm ảnh khỉ vọi đang đu cây

4. **dan-dieu.jpg** - Dân điểu hậu (Great Hornbill)

   - Tìm ảnh chim dân điểu với mỏ và sừng đặc trưng

5. **co-trang.jpg** - Cò trắng (Great Egret)
   - Tìm ảnh cò trắng đang săn mồi

## Lưu ý:

- Ảnh nên có độ phân giải tốt (ít nhất 800x600px)
- Format: JPG, PNG, hoặc WEBP
- Nên sử dụng ảnh có bản quyền hợp lệ hoặc ảnh miễn phí

## Nguồn ảnh gợi ý:

- Unsplash.com (ảnh miễn phí)
- Pexels.com (ảnh miễn phí)
- Wikimedia Commons (ảnh có bản quyền Creative Commons)
